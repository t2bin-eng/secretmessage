## 📋 사용 방법

### 교사용

1. `/admin` 페이지에서 **새 활동지 만들기** 클릭
2. 활동지 제목과 단원 입력
3. **엑셀 양식 다운로드** → 내용 작성 후 저장
4. 작성한 엑셀 파일 업로드
5. 단어판 크기 선택 (10×10 / 12×12 / 15×15)
6. **활동지 생성하기** 클릭
7. QR 코드 및 URL로 학생 공유

### 엑셀 양식 작성법

| word | clue | secret |
|------|------|--------|
| 을사늑약 | 1905년 일본이 외교권을 빼앗은 조약 | 너를 사랑해 |
| 갑오개혁 | 1894년 근대적 개혁 운동 | |

- **word**: 단어판에 숨길 단어 (공백 자동 제거)
- **clue**: 학생에게 보여줄 설명
- **secret**: 시크릿 메시지 (첫 번째 행에만 입력, 띄어쓰기로 단어 구분)
  - 예: `너를 사랑해` → 단어판에 `너를`과 `사랑해`가 각각 숨겨짐
  - 학생이 두 조각을 찾으면 시크릿 메시지 완성!

### 학생용

1. QR 스캔 또는 URL 직접 접속
2. 학번과 이름 입력 후 **활동 시작**
3. 설명 목록을 보고 단어판에서 단어 찾기
   - 시작 글자 클릭 → 끝 글자 클릭
4. 시크릿 메시지 조각도 단어판에서 찾기
5. 완료 후 **제출하기** 클릭

## 🛠 기술 스택

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **xlsx** — 엑셀 파일 파싱/생성
- **qrcode** — QR 코드 생성
- **localStorage** — 데이터 저장 (추후 Supabase 연동 가능)

---

## 📁 프로젝트 구조

```
src/
  app/
    admin/          교사 관리 페이지
      new/          새 활동지 만들기
      edit/[id]/    활동지 수정
      qr/[id]/      QR 코드 및 PDF
    play/[id]/      학생용 풀이 화면
  components/       재사용 UI 컴포넌트
  lib/              핵심 로직 (그리드 생성, 엑셀, PDF 등)
  types/            TypeScript 타입 정의
```

---

## 🔧 데이터 저장

현재 버전은 브라우저의 **localStorage**를 사용합니다.
- 같은 브라우저에서만 데이터 공유
- 브라우저 캐시 삭제 시 데이터 사라짐

추후 Supabase 연동으로 업그레이드 가능 (`src/lib/storage.ts` 수정)
